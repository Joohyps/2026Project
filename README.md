# Battle Ball — 1v1

Battle Ball(Steam) 스타일의 피구 게임을 Python + pygame으로 구현.  
영상에서 확인된 메커니즘을 그대로 재현.
니얼굴
---

## 실행 방법

```bash
pip install pygame
python main.py
```

PyCharm에서는 `main.py`를 열고 **Run** 버튼을 누르면 됩니다.

---

## 조작법

| | 이동 | 줍기 / 던지기 |
|---|---|---|
| **Player 1** (파란팀, 좌측) | `W` `A` `S` `D` | `Space` |
| **Player 2** (빨간팀, 우측) | `I` `J` `K` `L` | `U` |

- 공 근처에서 줍기 버튼 → **줍기**
- 공을 들고 줍기/던지기 버튼 → **던지기**
- 이동 방향키를 누르면 그 방향으로 조준
- `ESC` → 메뉴로 돌아가기

---

## 게임 규칙

- 상대를 공으로 **맞히면 1점**
- 맞히면 **1.8초 넘어졌다가 부활** (원위치)
- 부활 직후 **1.3초 무적 점멸**
- 공 4개 동시 진행
- **중앙선** 을 넘을 수 없음 (자기 진영에서만 이동)
- **2분 타이머** 종료 시 점수 높은 팀 승리

---

## 구현된 시각 효과

| 효과 | 설명 |
|---|---|
| 공 포물선 | z축 중력 시뮬레이션. `screen_y = game_y - z` |
| 그림자 분리 | 공이 높이 뜰수록 그림자가 멀어짐 |
| Y-sort 렌더링 | 아래쪽 오브젝트가 앞에 그려짐 |
| 중앙선 파티클 | 황금색 입자가 계속 올라옴 |
| 명중 이펙트 | 흰/금 파티클 폭발 |
| 공 소지 링 | 청록색 회전 링 |
| 넘어짐 애니메이션 | 옆으로 눕고 별이 돎 |
| 울타리 입체감 | 하단이 밝고 두꺼움, 상단이 어둡고 얇음 |

## 파일 구조

```
battle_ball/
├── main.py        엔트리 포인트
├── settings.py    모든 상수 (물리, 색상, 크기)
├── game.py        게임 상태머신 + 렌더링 + 충돌 판정
├── court.py       코트 배경 (정적 pre-render)
├── player.py      플레이어 클래스
├── ball.py        공 클래스 (z축 물리)
├── effects.py     파티클 시스템
└── requirements.txt
```
